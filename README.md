**DGQuestionnaire - A Digital Arrest Verification Tool**

**Project Overview:**
DGQuestionnaire is a high-fidelity web-based diagnostic tool designed to help users—particularly senior citizens—identify and respond to potential Digital Arrest scams, an increasingly common form of cybercrime in India.

The tool guides users through a short, structured 5-question assessment that evaluates suspicious caller behaviour. Based on the user’s responses, the system generates an instant scam risk assessment, allowing individuals to quickly determine whether the situation may be fraudulent.

By combining clear guidance with an accessible interface, DGQuestionnaire aims to increase public awareness of cybercrime and empower vulnerable users to take safer digital actions.


**Technical Implementation:**
The prototype was developed using HTML5, CSS3, and JavaScript, and coded entirely within TextEdit.

**Application Architecture:**
Built as a Single Page Application (SPA) to enable smooth navigation and instant transitions between questions.

**Performance Optimisation:**
Designed to function effectively on low-bandwidth connections (2G/3G networks) to ensure accessibility across diverse user environments.

**Visual Interface:**
Features a custom SVG Risk Meter that visually represents scam probability.
The meter includes a mathematically defined pivot point (130,120) to ensure precise needle rotation and accurate visual feedback.

**Mobile Accessibility:**
Interface layout constrained to a maximum width of 400px, ensuring optimal readability on mobile devices.
Typography set to 18px or larger to support senior users and improve accessibility.

**Touch Interaction:**
All interactive buttons have a minimum height of 55px, exceeding the standard 44px touch-target guideline, making interaction easier for users with reduced dexterity.
Key Features

**Accessibility:**
The interface is designed with senior users in mind, using large text, simple navigation, and clear interaction elements.

**Safety:**
The platform does not collect or store personal information, and never requests sensitive data such as Aadhaar numbers, OTPs, or banking details.

**Environmental Considerations:**
DGQuestionnaire provides a fully digital awareness solution, reducing reliance on paper-based materials such as posters or pamphlets.

**Functionality:**
The system calculates and displays a Scam Probability Percentage, supported by a 180° rotating risk meter (-90° to +90°) for intuitive visual interpretation.

**Safety Guide:**
The project includes an integrated Safety Guide that educates users about Digital Arrest scams and how to respond safely.
The guide explains:
- What Digital Arrest scams are and how scammers create fear and urgency
- Common scam tactics, including fake documents, uniforms, and staged investigations
- Warning signs such as demands for money, bank information, or OTP verification
- Recommended actions, including:
    - Immediately ending suspicious calls
    - Never sharing OTPs or financial details
    - Reporting incidents to India’s National Cybercrime Helpline (1930)
